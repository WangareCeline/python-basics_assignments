# python-basics_assignment
A comprehensive collection of Python scripts covering the essential building blocks of programming. This repository serves as a reference for core concepts, ranging from simple syntax to Object-Oriented Programming (OOP).

## Contents

### 1. Variables & Data Types
* Understanding **Strings, Integers, Floats, and Booleans**.
* Type conversion and dynamic typing in Python.

### 2. Data Structures
* **Lists**: Creating, slicing, and using list comprehensions.
* **Sets**: Handling unique collections and mathematical set operations.
* **Dictionaries**: Key-value pair management.

### 3. Control Flow & Logic
* **Conditional Statements**: `if`, `elif`, and `else` logic.
* **Logical Operators**: Combining conditions using `and`, `or`, and `not`.

### 4. Loops & Iteration
* **While Loops**: Running code until a specific condition (like a "stop" command) is met.
* **For Loops**: Iterating over sequences and ranges.
* **Loop Control**: Practical use of `break` and `continue`.

### 5. Functions
* Defining reusable code blocks with `def`.
* **Scope**: Understanding local vs. global variables.
* **Nested Functions**: Calling functions from within other functions.

### 6. Object-Oriented Programming (OOP)
* **Classes & Objects**: Creating blueprints and instantiating objects.
* **Encapsulation**: Using private attributes (`__variable`) and Getters/Setters.
* **Composition**: Building complex objects (like a `Cylinder`) from simpler ones (like a `Circle`).
* **Interaction**: Comparing different objects using class methods.
